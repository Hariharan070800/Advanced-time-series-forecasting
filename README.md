# Advanced Time Series Forecasting with Attention

This project demonstrates multi-step time series forecasting using
Seq2Seq LSTM with Attention mechanism implemented in Python.

## Tools Used
- Python
- Jupyter Notebook
- TensorFlow / Keras
- Pandas, NumPy, Matplotlib

## Model
- Encoder Decoder LSTM
- Attention Mechanism
- Walk-forward validation
